# Eulecast
Podcast

Install dependencies
```sh
$ npm install
```

### Gulp tasks
browser sync server
```sh
$ gulp serve
```

build application
```sh
$ gulp build
```

### REACT + FLUX

* [React](https://facebook.github.io/react/) - A javascript library for building user interfaces
* [Flux](https://facebook.github.io/flux/) - Application architecture for building user interfaces
